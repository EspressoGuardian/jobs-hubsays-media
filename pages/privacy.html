Privacy

Hubsays is designed as a low-data public beta. There are no user accounts, no resume uploads, no application forms, and no candidate profile warehouse in this release.

We do not intentionally collect, sell, or profile personal data through this site. The product is built as a static browse-and-click layer with direct-source links and local-only preference storage.

  • No user accounts in v1.
  • No resume uploads or candidate profiles in v1.
  • No on-site application flow in v1; users should always apply at the original source.
  • No behavioural ad stack, no social trackers, and no third-party analytics scripts in the site code.
  • Decision Desk preferences stay in the browser on the local device only.
  • We do not operate a user database for this beta.
Important technical boundary: standard infrastructure providers such as GitHub Pages, Cloudflare, or the visitor's network path may necessarily process IP addresses transiently to deliver the site. Hubsays does not use that infrastructure to build visitor profiles or store application data.
